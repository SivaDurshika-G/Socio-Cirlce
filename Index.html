🌍 SocioCircle

"Where Surplus Meets Need"

🔹 Problem

Every day, tons of food and usable items are wasted in households, hostels, restaurants, and events, while at the same time, millions of underprivileged people struggle without basic needs. 👉 There is no structured platform that connects donors and NGOs in real-time for smooth pickups and distribution.

🔹 Solution – Our Startup

SocioCircle is a mobile & web app that acts as a bridge between donors, NGOs, and volunteers.

  • 🍲 Donors can post surplus food, clothes, books
  • 🤝 NGOs/volunteers can claim pickups & distribute
  • 📍 Live pickup tracking like Swiggy/Zomato
  • 📊 Impact dashboard shows contributions

🔹 Tech Stack

  • ⚛️ React.js (CRA) → Frontend UI
  • 🌐 Node.js + Express → Backend API
  • 🗄 MongoDB → Store data
  • 🔑 JWT → Authentication
  • 🎨 CSS3 + Animations → UI/UX
  • 📧 Nodemailer → Notifications

🔹 Key Features

  • ✅ Easy Donations – Food, clothes, books
  • ✅ NGO Matching System
  • ✅ Live Pickup Tracking
  • ✅ Impact Dashboard
  • ✅ Notifications & Email Alerts
  • ✅ Gamification & Badges

🔹 Social Impact

🌾 Reduces food waste 💙 Provides essentials to underprivileged 🤝 Encourages sharing in society 🌍 Builds sustainable ecosystems

🔹 Why This Idea is Special

Unlike general donation apps, SocioCircle is focused on real-time action.

Donor → Posts item → NGO claims → Live pickup → Society benefits immediately 🚀
